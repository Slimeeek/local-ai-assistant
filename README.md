# 🤖 Local AI Assistant

**Локальный ИИ-ассистент с возможностью смены моделей**

Полностью автономный ассистент, работающий на вашем компьютере без облачных сервисов. Меняйте модели ИИ как "мозги" помощника, сохраняя всю память и настройки.

![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)
![Platform](https://img.shields.io/badge/Platform-Linux-orange.svg)

##  Возможности

- 🧠 **Смена моделей на лету** — переключайтесь между разными LLM без потери данных
-  **Долгосрочная память** — ассистент помнит ваши разговоры и предпочтения
- 🖥️ **Локальный GUI** — удобный графический интерфейс без веба
- 🔒 **Полная приватность** — все данные остаются на вашем компьютере
- 🎯 **Настраиваемая личность** — меняйте характер ассистента через файлы конфигурации

## 📋 Системные требования

### Минимальные:
- **ОС:** Linux (Ubuntu 20.04+, Debian 11+, Fedora 36+)
- **Python:** 3.10 или выше
- **RAM:** 8 GB
- **Диск:** 10 GB свободного места
- **GPU:** Опционально (NVIDIA с CUDA для ускорения)

### Рекомендуемые:
- **RAM:** 16 GB
- **GPU:** NVIDIA с 4+ GB VRAM (RTX 3050 и выше)
- **CPU:** 4+ ядер

## 🚀 Быстрый старт

### Способ 1: Автоматическая установка (рекомендуется)

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Slimeeek/local-ai-assistant.git
cd local-ai-assistant

# 2. Запустите установщик
chmod +x install.sh
./install.sh

# 3. Запустите ассистента
./run.sh
